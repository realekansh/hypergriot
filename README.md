# HyperGriot

### A modular Telegram moderation & community bot,
#### This repository contains the bot core, modules, workers, and deployment config.

##### Quickstart (dev)
1. Copy .env.example to .env and fill values
2. docker compose up --build
3. Run migrations: make migrate
4. Start bot: make run
